POST /api/generate

Deskripsi:

Generate QR code QRIS dengan jumlah nominal tertentu. Hasil berupa URL gambar QR yang dapat langsung digunakan.

Request Body (JSON):

{
  "amount": 10000
}

Response (200):

{
  "qris": "https://yourdomain.com/qris/QR1690645589651.jpg"
}

Response (400):

{
  "error": "Invalid amount"
}

Response (500):

{
  "error": "Failed to generate QRIS"
}
GET /qris/:filename

Deskripsi:

Mengambil file QRIS (.jpg) dari folder /tmp.

Contoh URL:

GET /qris/QR1690645589651.jpg

Response:

File image QRIS (.jpg).

Response (404):
